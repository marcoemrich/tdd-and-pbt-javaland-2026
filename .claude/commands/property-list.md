# Property List Command

**MANDATORY**: Use the Task tool with `subagent_type: "property-list"` to create the property list.

Do NOT perform this phase manually. The agent systematically identifies properties and creates a comprehensive list.

Provide the agent with the necessary context:
- Feature/function to implement
- Target properties file path (e.g., `*Properties.java`)
- Implementation file path
- Any constraints or requirements from the user
